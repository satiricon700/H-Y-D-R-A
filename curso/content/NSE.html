Scripts NSE de Nmap - Automatización y Personalización de Escaneos

Los scripts NSE (Nmap Scripting Engine) son una característica poderosa de Nmap que permite la automatización y personalización de escaneos de red. En esta página, te explicaremos en detalle qué son los scripts NSE, cómo utilizarlos y cuándo son útiles, además de proporcionar ejemplos y comandos de uso.


¿Qué son los Scripts NSE?

Los scripts NSE son pequeños programas escritos en lenguaje Lua que se ejecutan dentro de Nmap. Estos scripts proporcionan funcionalidades adicionales a las exploraciones de red, como detección de vulnerabilidades, recopilación de información y más. Los scripts NSE están diseñados para facilitar la personalización de los escaneos y automatizar tareas comunes de seguridad de red.


Cómo Utilizar los Scripts NSE

Para utilizar los scripts NSE en Nmap, sigue estos pasos:

  1. Selecciona un objetivo para tu escaneo. Puedes especificar una dirección IP, un rango de IP o un nombre de host.
  2. Abre una terminal o línea de comandos y utiliza el siguiente formato de comando para ejecutar un escaneo con scripts NSE: 
nmap -sC -sV [objetivo]

Donde:

  • -sC: Habilita la ejecución de scripts NSE.
  • -sV: Detecta versiones de servicios en puertos abiertos.
  • [objetivo]: Especifica la dirección IP o nombre de host del objetivo.

Ejemplos de Uso

A continuación, algunos ejemplos de comandos Nmap con scripts NSE:

nmap -sC -sV 192.168.1.1

Este comando ejecuta un escaneo en la dirección IP 192.168.1.1 con la detección de versiones de servicios y la ejecución de scripts NSE. 

nmap -sC -p 80,443 192.168.1.1

Este comando escanea los puertos 80 y 443 en la dirección IP 192.168.1.1 con la ejecución de scripts NSE.

Cuándo Utilizar los Scripts NSE

Los scripts NSE son útiles en diversas situaciones, como:

  • Descubrimiento de servicios y puertos abiertos.
  • Detección de vulnerabilidades en servicios.
  • Enumeración de información de sistemas y servicios.
  • Automatización de tareas de seguridad.

Los scripts NSE son flexibles y se pueden adaptar a tus necesidades específicas en la exploración y auditoría de redes.

Ejemplos de Scripts NSE

Estos son algunos ejemplos de scripts NSE disponibles en Nmap:

  • http-title.nse: Muestra el título de la página web en puertos HTTP.
  • ssl-enum-ciphers.nse: Enumera las suites de cifrado SSL/TLS disponibles en un servidor.
  • ftp-anon.nse: Comprueba si un servidor FTP permite conexiones anónimas.
  • vuln.nse: Detecta vulnerabilidades conocidas en servicios y aplicaciones.

Estos son solo algunos ejemplos, y hay muchos más scripts disponibles para diversas tareas de exploración y auditoría de red.
